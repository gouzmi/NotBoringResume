# NotBoringResume

**NotBoringResume** is a web app that lets you build a clean, professional resume in minutes — without the hassle of word processors or overpriced tools.

Fill in your details, pick a template, and get a ready-to-use resume instantly.

## Features

- Simple and intuitive form-based editor
- Live preview as you type
- Export-ready output

## Templates

Currently, one template is available: **Lean** — a clean, minimal design that keeps the focus on your content.

More templates are coming in the future.

## Tech Stack

- [React](https://react.dev) + [TypeScript](https://www.typescriptlang.org)
- [Vite](https://vite.dev)
- [Tailwind CSS](https://tailwindcss.com)
